Spec da barra fina (tela cheia)

Conteúdo
  • Título (esq., elipse)
  • Tom + posição 2/5 (dir.)
  • Barra de progresso 3px
  • Sem botões (← ‹ › ⚙ ⛶)
Layout
  • sticky; top:0
  • linha: título flex:1 + meta flex:none
  • progresso full-width, embaixo
  • fundo glass + blur + borda
Tipografia
  • Título: Inter 13px / 600
  • Tom: Mono 12.5px / 700 · --accent
  • Posição: Mono 12.5px · --muted
Alturas
  • padding 6px 14px 0 + safe-area-top
  • linha de texto ~18px
  • progresso 3px
  • total ≈ 28–32px (era ~69px)
Implementação
  • tela cheia deixa de display:none a barra
  • esconde só os .pbtn/.pbtn-sm
  • injeta o Tom no meta (drawPlayer)
  • #pv-exitfull segue p/ sair
Palco / a11y
  • Tom em acento = glance rápido
  • contraste mantido (sobre glass)
  • sem alvo de toque na barra (ok: sem botão)
  • respeita safe-area (notch)

Obs.: o app usa Inter (UI) e JetBrains Mono (Tom/cifra) de verdade; aqui o preview cai pra fontes de sistema (CDN bloqueado no artifact), mas as proporções/tokens são os reais do Louvai.
